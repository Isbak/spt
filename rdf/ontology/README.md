# Ontology

Domain-neutral core ontology files will be added here.


## Ontology Versioning

Ontology assets use semantic versioning:

- Major version: breaking semantic change.
- Minor version: new concepts.
- Patch version: non-breaking corrections.

Each ontology should declare `owl:versionInfo`, `dcterms:created`, `dcterms:modified`, `dcterms:creator`, and `dcterms:publisher`.
