# ADR-0001: Platform Scope

## Status
Accepted

## Context
The platform must support semantic modeling, validation, querying, provenance, governance, visualization and future AI/agent integration.

## Decision
Build a generic semantic platform using RDF, OWL, SHACL, SPARQL, Fuseki, Flask, Python and Azure DevOps.

## Consequences
The platform is reusable across domains and avoids embedding domain-specific logic.
