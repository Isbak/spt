# Governance Model

## Concepts

- Data Owner
- Data Steward
- Custodian
- Policy
- Classification
- Retention Rule
- Access Rule
- Approval Status

## Rules

- Every ontology must have an owner.
- Every vocabulary must have a steward.
- Every dataset must have provenance.
- Every production graph must have classification metadata.
- Agents write only to sandbox or integration graphs unless explicitly approved.
